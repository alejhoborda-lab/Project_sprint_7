# Análisis de Vehículos Usados - Aplicación Web

## Descripción del proyecto
Esta aplicación web interactiva permite explorar y analizar datos de vehículos usados en Estados Unidos. Desarrollada con Streamlit, proporciona visualizaciones dinámicas para ayudar a usuarios a entender tendencias del mercado automotriz.

## Funcionalidades
- **Histograma interactivo**: Visualiza la distribución del odómetro (recorrido) de los vehículos
- **Gráfico de dispersión**: Explora la relación entre precio y odómetro
- **Controles interactivos**: Casillas de verificación para mostrar/ocultar gráficos
- **Interfaz intuitiva**: Navegación fácil y visualizaciones claras

## Cómo usar la aplicación
1. Abrir el siguiente enlace: https://project-sprint-7-dovm.onrender.com
2. Utiliza las casillas de verificación para mostrar diferentes gráficos
3. Explora los datos de vehículos usados de forma interactiva
